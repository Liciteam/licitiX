Despre proiect

Proiectul consta in conceperea unei aplicatii Web destinate realizarii licitatiilor de produse cu caracter personal, asemanatoare celor de pe eBay.

Alegere Proiect

Data alegerii proiectului : 28.03.2016. (Finalizat)

In urma discutiei intre membrii echipei , s-a hotarat 20.AuctioX.

Stabilirea obiectivelor de proiectare

Aplicatia este realizata pentru a facilita urmatoarele :

- Vizualizarea produselor licitate ( cautare cu filtre )

- Vizualizarea meta-datelor unui produs

- Introducerea de obiecte spre a fi licitate

- Realizarea licitatiilor printr-un sistem de plata real

- Sugestii de produse pe baza preferintelor utilizatorului
